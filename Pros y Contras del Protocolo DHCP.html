Ventajas y desventajas del DHCP

Ventajas de los servidores DHCP Ventajas: 1- Únicamente se configura el servidor que entregara de un rango de direcciones una IP a cada cliente. 2- Facilidad de configuración. 3- Las direcciones son controladas por el servidor, lo cual logra su facilitación al momento de dar seguimiento y supervisar. 4- En una red de bastantes usuarios, el servidor DHCP es necesario para asignación de IP de forma automática. 5- Se entregan todos los parámetros básicos de TCP-IP

Desventajas de los servidores DHCP

Desventajas: 1- En caso de que se requiera un DNS, es necesario configurarlo manualmente para indicar las direcciones IP correspondientes. 2- En caso de que el servidor DHCP falla, todas las maquinas clientes renovaran su IP al no obtener respuesta alguna, lo cual provocara que toda la red se detenga.
